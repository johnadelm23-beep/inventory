# 📦 Inventory App

A Flutter-based Inventory Management System with Firebase integration.  
Supports authentication, product management, user roles (Admin/User), and real-time updates.

---

## 🚀 Features

- 🔐 Firebase Authentication (Login / Register)
- 📦 Add / Edit / Delete Products
- ⚠️ Low stock warning system
- 👤 User management (Admin panel)
- 📊 Dashboard with statistics
- ☁️ Firestore real-time database
- 📱 Responsive UI (Mobile + Desktop ready)
- 🖥️ Windows desktop support
